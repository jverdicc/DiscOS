# Support

## Getting help

- For usage questions and troubleshooting, open a GitHub Discussion or Issue.
- For confirmed bugs, use the bug report issue template.
- For feature ideas, use the feature request template.
- For security concerns, follow [`SECURITY.md`](./SECURITY.md).

## What to include in support requests

- DiscOS version or commit SHA
- OS/environment details
- Exact command(s) run
- Full output/error text
- Minimal reproduction steps

## Maintainer response expectations

Maintainers will triage incoming requests as capacity allows. High-quality,
reproducible reports are resolved fastest.
