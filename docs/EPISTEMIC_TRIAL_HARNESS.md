# Epistemic Trial Harness (DiscOS wrapper)

DiscOS is the workload runner/client side for the Epistemic Trial Harness.

Canonical harness documentation lives in EvidenceOS:
- <https://github.com/jverdicc/EvidenceOS/blob/main/docs/EPISTEMIC_TRIAL_HARNESS.md>

Blessed analysis pipeline entrypoint:
- <https://github.com/jverdicc/EvidenceOS/blob/main/docs/TRIAL_HARNESS_ANALYSIS.md>

## DiscOS role in trial execution

Use DiscOS to:
- Create deterministic claim artifacts and commit manifests.
- Run adaptive workload sequences against an EvidenceOS daemon.
- Capture execution outputs and capsule fetches for downstream analysis.
